Jag använde en dual-axis plot för att jämföra oljepris och USD-index över tid eftersom de har olika skalor.
Typ: Regression
Dataset: Brent Oil Prices och USD index


Modellen uppnådde ett R² på 0.57 vilket indikerar att den förklarar en måttlig del av variationen i oljepriset.


Modellen fångar generella trender men har svårt att förutsäga extrema prisrörelser.
